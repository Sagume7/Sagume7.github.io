RESTful

Restful API 是一个广泛使用的软件接口设计模式，其核心思想是通过 HTTP 标签（GET、POST、PUT、DELETE 等）实现对象间的通信。

常见接口类型

  1. GET： 获取资源信息
  2. POST： 创建新资源
  3. PUT： 更新资源状态
  4. DELETE： 删除资源

标签举例：

API Route
GET /api/users
POST /api/users
PUT /api/users
DELETE /api/users
